# Web Project

This directory contains the HTML/CSS/JavaScript web project for GitOps.

## Structure

```
web/
├── index.html           # Main HTML file
├── src/
│   ├── css/
│   │   └── styles.css   # Main stylesheet
│   ├── js/
│   │   └── script.js    # JavaScript functionality
│   └── assets/          # Images, fonts, and other assets
```

## Running the Project

You can open `index.html` directly in a web browser or serve it through a local web server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (if http-server is installed)
npx http-server

# Then visit http://localhost:8000 in your browser
```

## Features

- Responsive design
- Smooth scrolling navigation
- Interactive button with event listeners
- Professional styling with gradients and transitions
